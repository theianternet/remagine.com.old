Check / Withdrawal Entry

Enter new withdrawals by selecting the "Withdrawal" tab below the register, and clicking on "New".  You can also quickly create withdrawals of type Debit or Check by selecting them from the Transaction menu.

If you wish to print a transaction on pre-printed checks after entering it, select "Print" as the check number to add it to the print queue.  Later, select "Print Checks" from the "File" menu to print a batch of queued checks.
