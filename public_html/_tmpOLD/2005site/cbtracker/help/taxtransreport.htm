Tax Related Transactions XML Report

All tax related transactions for a given date range can be exported to XML.  This allows you to import the data into another program, manipulate it with a scripting language, or pretty much whatever you want to do with it.  I've also included a stylesheet that will do a nice job formatting it if you just want a simple transaction report.  Access this report from the Reports menu.  When the report is generated, your web browser will open to display the report.
