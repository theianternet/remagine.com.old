Creating a file

So you want to manage your finances do you?  Well, you've come to the right place.  CheckBook Tracker has been designed to make it as easy as possible.

When you first open CBT (CheckBook Tracker), you'll be presented with a fairly sparse user interface but only because no file is currently open.

Your first step is to create a new file by selected "NEW" from the "FILE" menu.  Name the file anything you like.  There is not file extension added, so I tend to use "cbt.file" or maybe you will want to use "checkbook.cbt"  Don't worry, you won't have to remember this filename every time you open CBT.  From now on, the last file you used will automatically be opened at startup.
