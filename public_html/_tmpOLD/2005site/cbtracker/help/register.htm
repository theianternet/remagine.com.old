The Transaction Register

Most of your time will probably be spent staring at the transaction register.  The transaction register has several view options that affect how your transactions are displayed.

You can choose to view only transactions that have not been reconciled (or all transactions) from the "View" menu.   There, you can also select to display the memo field for each transaction.

Clicking on the "Num" or "Date" header line at the top of the transaction register will sort the displayed transactions by either transaction number or date.

When you click on a transaction line, the selected transaction will be displayed at the bottom of the screen.

If you double-click on a transaction line, the selected transaction will be displayed at the bottom of the screen, and it will go into edit mode.

Right clicking on a transaction line allows you to change the transaction status (Blank, Cleared, Reconciled, Void) or Edit and Delete.
